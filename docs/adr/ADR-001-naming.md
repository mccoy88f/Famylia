# ADR-001: Naming prodotto

**Stato:** Accettato  
**Data:** 2026-05-28

## Contesto

La specifica v3 usa "Family Hub"; il repository GitHub è **Famylia**.

## Decisione

Usare **Famylia** per UI, PWA manifest, package Dart (`famylia_server`, `famylia_app`) e documentazione utente.

## Conseguenze

- Aggiornare gradualmente riferimenti "Family Hub" nella spec
- Database e variabili Docker usano `famylia` come nome
